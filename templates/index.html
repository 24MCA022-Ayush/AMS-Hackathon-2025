Problem Statement: Sum of Two Numbers

Write a program that reads two integers from standard input and outputs their sum to standard output.

Input Format:

Two space-separated integers A and B.

Output Format:

A single integer representing the sum of A and B.

Constraints:
  • -1000 ≤ A, B ≤ 1000
Sample Input:
5 7
Sample Output:
12
Allowed Languages:
  • C (.c)
  • C++ (.cpp)
  • Java (.java) - Must contain a public class matching the filename
  • Python (.py)
Maximum file size: 1MB. Accepted file types: .c, .cpp, .java, .py
